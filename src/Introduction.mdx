# Antd Inline Edit

Antd Inline Edit is a customizable React component that allows users to edit text in-place. It is built using the Ant Design UI library and can be easily integrated into any React project.

## Features

- Edit text in-place
- Customizable display rendering
- Callback function on save

## Installation

To install Antd Inline Edit in your project, simply run:

```
npm install @bukku/antd-inline-edit
```

## Usage

To use Antd Inline Edit in your React project, simply import it and add it to your component:

```jsx
import React from 'react';
import InlineEdit from '@bukku/antd-inline-edit';

function MyComponent() {
  const handleSave = (value) => {
    console.log('Saved value:', value);
  };

  return (
    <InlineEdit
      initialValue="Input shows this on click"
      defaultValue="Input reverts to this on undo"
      inputMaxLength={100}
      onSave={handleSave}
      displayRenderer={(value) => (
        <div style={{ fontWeight: 'bold' }}>{value}</div>
      )}
    />
  );
}
```
